版本：v1.0

## 1. 系統啟動與初始化

### 電源 ON

- Power Check
- 自我檢測
- 讀取設定參數

### 初始化模組

- RTC 時鐘
- 按鍵 / IO
- 顯示驅動
- 震動 / 蜂鳴器
- 藍牙（BLE）模組
- 儲存（SPI Flash）
- 聲音系統（I2S + SD）
- 記憶體檢查

### 模式判斷

- 是否為主機模式？
  - 是 → 進入主機模式（可連線）
  - 否 → 進入副機模式（等待配對）

### 待機畫面

- 顯示：
  - 時間
  - 電量
  - 模式
  - 連線狀態

## 2. 主流程（任務執行循環）

### 待機狀態（IDLE）

- 顯示待機畫面
- 等待操作

### 開始任務（長按主鍵 ≥ 2 秒）

- 建立新任務
- 記錄任務開始時間
- 同步時間
- 開始計時
- 切換至執行模式

### 任務執行中（RUNNING）

- 計時運行
- 定時提醒（6 秒循環）
- 按鍵操作
- 記錄數據
- 顯示資訊

#### 節律提醒循環（每 6 秒）

1. 提醒開始（BEEP / 震動）
2. 6 秒倒數（節拍器）
3. 提醒結束

#### 每 4 分鐘檢查

- 是否需要高頻提醒？
  - 是 → 強提醒（聲音 + 震動 + 提示）

### 任務暫停（PAUSE）

- 暫停時間與提醒
- 保持資料
- 顯示暫停狀態

#### 判斷

- 是否繼續任務？
  - 是 → 回 RUNNING
  - 否 → 結束任務

### 結束任務（長按主鍵 ≥ 2 秒）

- 停止計時
- 停止提醒
- 儲存任務資料
- 寫入 Flash
- 回到待機

### 背景處理

#### 異常處理

- 按鍵長按處理
- 過早中斷處理
- 錯誤異常
- 儲存異常
- 低電量顯示

#### 資料即時處理

- 按鍵事件
- 時間戳記
- 同步資料
- BLE 傳輸
- 定期寫入 Flash

## 3. 按鍵與事件處理邏輯

### 主鍵（正下方大鍵）

#### 短按（< 1.5s）

- 記錄事件（依模式）
  - 給藥事件（時間記錄 + 提示）
  - 通氣事件（時間記錄 + 提示音）

#### 長按（≥ 2s）

- 開始 / 結束任務
- 暫停 / 繼續任務

### 左側上下鍵（功能切換）

- 切換模式：
  - 給藥模式（預設每 4 分鐘提醒）
  - 通氣模式（每 6 秒節拍）
- 更新顯示 + 提示音

### 右側鍵（電源）

- 短按（< 1.5s）：螢幕亮 / 滅
- 長按（≥ 2s）：開機 / 關機（含安全判斷）

### 左上轉角鍵（靜音鍵）

- 短按：標記靜音事件
- 長按（≥ 2s）：切換靜音 ON / OFF

## 4. 主副機通訊流程（BLE）

### 主機（中心）

- 開機 → 進入主機模式
- 廣播（可被發現）

#### 掃描流程

- 搜尋副機
- 找到設備？
  - 否 → 持續搜尋
  - 是 → 建立連線（配對）

#### 同步內容

- 時間同步
- 事件同步
- 指令同步

#### 連線維持

- 心跳檢測

### 副機（周邊）

- 開機 → 進入副機模式
- 等待連線

#### 收到連線請求？

- 否 → 持續等待
- 是 → 配對連線

#### 同步資料

- 接收主機同步
- 資料接收 / 傳送
- 事件回報

### BLE 傳輸資料內容

- 事件資料（時間戳、類型）
- 裝置狀態（電量、模式）
- 設定參數
- 指令（開始 / 停止 / 切換）

### 連線狀態顯示

- 已連線
- 連線中
- 斷線

## 5. 資料記錄與儲存

### 流程

1. 事件產生（按鍵 / 系統）
2. 加入事件緩衝區（RAM）
3. 暫存（約 5 秒）
4. 寫入 SPI Flash（加密）
5. 標記已寫入
6. 可透過 BLE / Wi-Fi 傳輸至 APP

### 資料結構（單筆事件）

| 欄位 | 說明 |
|------|------|
| Event ID | 事件編號 |
| Timestamp | 時間 |
| Event Type | 類型（給藥 / 通氣 / 其他）|
| Source | 來源（按鍵 / 系統）|
| Device ID | 裝置 |
| Mode | 模式 |
| Extra Data | 備註 |
| Sync Flag | 是否同步 |

### 儲存位置

- SPI Flash（主要）
- SD Card（選配）

---

## 6. APP 資料流程

1. 裝置傳輸（BLE / Wi-Fi）
2. APP 接收
3. 資料解析與驗證
4. 本地儲存（SQLite）
5. 資料呈現（時間軸 / 清單 / 圖表）
6. 匯出（PDF / CSV / EMS 系統）
7. 備份 / 上傳雲端

## 7. 硬體架構

### 電源模組

- Li-ion Battery（3.7V）
- TP4056 充電保護板
- 電源管理（穩壓 / 升降壓）
- 輸出：
  - 3.3V
  - 5V

### 主控晶片

- ESP32-S3

### 輸入模組

- 主鍵（下方）
- 左側上下鍵
- 右側電源鍵
- 左上靜音鍵

### 輸出模組

- LCD / OLED 顯示
- 震動馬達
- 蜂鳴器

### 通訊模組

- BLE 5.0（主 / 副機）
- Wi-Fi（選配）

### 儲存模組

- SPI Flash
- SD Card

### 音訊模組

- I2S + 喇叭（選配）

### 介面模組

- USB-C（充電 / 資料）
- Pogo Pin（燒錄 / 測試）
